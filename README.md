# AyuGram & Telegram Session Manager

Удобный лаунчер на базе PowerShell для управления множеством сессий Telegram/AyuGram с поддержкой прокси.

## 📁 Структура проекта

*   `auygram.bat` — Основной файл для запуска.
*   `auygram.ps1` — Скрипт логики (автопоиск сессий, управление прокси, меню).
*   `TgWsProxy_windows.exe` — Прокси-сервер (должен находиться в корне или на рабочем столе).
*   **Папки сессий** (например, `Сергей`, `Анна` и т.д.) — Каждая папка содержит свой `Telegram.exe` или `AyuGram.exe` и данные `tdata`.

## 🚀 Как пользоваться

1.  **Запуск:** Просто запустите `auygram.bat`.
2.  **Выбор сессии:** Введите номер нужной сессии из списка и нажмите `Enter`.
3.  **Управление прокси:** 
    *   Программа автоматически проверяет, запущен ли `TgWsProxy_windows.exe`.
    *   Если прокси выключен, он запустится автоматически при выборе любой сессии.
    *   Клавиша `P` в меню позволяет принудительно перезапустить прокси.

## ➕ Добавление новых аккаунтов

Чтобы новая сессия появилась в списке:
1.  Создайте в корне проекта новую папку с любым именем (например, `Работа`).
2.  Поместите в неё файлы Telegram (обязательно наличие `Telegram.exe` или `AyuGram.exe`).
3.  При следующем запуске лаунчера сессия «Работа» автоматически появится в списке.

## 🚀 Перенос на другие ПК

Проект полностью портативен. Для переноса:
1. Скопируйте всю папку проекта на другой ПК.
2. Убедитесь, что установлена Windows 7 SP1 или выше (где есть PowerShell 5.1).
3. Запустите `auygram.bat`. Все сессии подтянутся автоматически.

## 🛠 Технические требования
*   **ОС:** Windows 7/8/10/11
*   **Среда:** PowerShell 5.1+
*   **Библиотеки:** .NET Framework 4.5+ (встроено в современные Windows)

## ❓ Решение проблем

*   **Сессия не видна в списке:** Проверьте, что внутри папки с именем сессии есть файл `Telegram.exe` или `AyuGram.exe`.
*   **Ошибка при запуске .bat:** Убедитесь, что `auygram.ps1` находится в той же папке, что и `auygram.bat`.
*   **Прокси не запускается:** Убедитесь, что файл `TgWsProxy_windows.exe` не заблокирован антивирусом.
